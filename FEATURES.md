# 功能特性清单

本文档列出所有已实现功能和计划功能。

---

## ✅ 已实现功能

### 核心调度优化
- [x] 两种活动类型支持：A类（需要专注，单任务）/ B类（可并行，多任务）
- [x] 两种优化策略：最大收益填满时间 / 均衡模式留出休息时间
- [x] 自动计算切换活动的时间开销（15-20分钟）
- [x] 文件自动监听，数据变化自动重新计算推荐
- [x] 单元测试全覆盖

### 活动统计模块
- [x] 多角色支持，每个角色独立统计
- [x] 支持两种活动：搬砖（银币收益）/ 蹲星（成功计数）
- [x] 进度条显示目标完成百分比
- [x] 每日记录表格展示
- [x] 添加/删除记录
- [x] 设置目标银币/时长/总收入
- [x] 集成活动推荐算法，根据剩余目标计算每日安排
- [x] 全局建议设置可配置：每日总时长、最大并发、切换时间

### 数据分析模块 (PySide6 新增)
- [x] 每日收益趋势图（当日收益 + 累计收益两条曲线动画绘制）
- [x] 各角色累计收益对比柱状图（柱状从0增长动画）
- [x] 总活动时长按活动类型分布饼图（旋转展开动画）
- [x] 总活动时长按角色分布饼图（超过8个自动合并小项到"其他"）
- [x] 所有图表深色主题，和应用风格一致
- [x] 重新加载数据 / 重新播放动画按钮

### 存储时长统计
- [x] 多个窝子/钓点累计时长统计
- [x] 随时增加/减少时长
- [x] 添加/删除角色
- [x] 表格展示剩余时长

### 饵料/钓具库存
- [x] 记录每个饵料/钓具的总购买量、总使用量
- [x] 实时计算剩余数量
- [x] 增加库存 / 使用库存操作
- [x] 库存不足提示，确认是否使用全部剩余
- [x] 汇总统计：总品类数、总剩余数量

### 转盘抽奖
- [x] 自定义奖项，每个奖项可配置名称、概率、颜色
- [x] 转盘旋转动画，减速模拟真实物理
- [x] 旋转停止自动计算中奖奖项
- [x] 奖项表格编辑，添加/删除奖项
- [x] 总概率校验，提示不等于100%

### 账号密码管理
- [x] 多账号管理
- [x] **Fernet 对称加密**存储密码，本地安全
- [x] 界面固定显示 `******`，不暴露密码长度
- [x] 一键复制密码到剪贴板
- [x] 添加/删除/修改密码
- [x] 加密密钥单独存储

### 桌面悬浮提醒
- [x] 独立圆形窗口，可拖动
- [x] 不显示在任务栏，不占用任务栏位置
- [x] **点击不抢游戏焦点**，保持游戏焦点不变
- [x] 气泡提醒，尖端紧贴圆形窗口
- [x] 气泡浮动动画，柔和上下浮动
- [x] 每日定时提醒：
  - 00:00 提醒打开加班机
  - 18:00 提醒开播
  - 饭点提醒吃饭
  - 每两小时提醒休息
- [x] 自定义提醒，支持 N 分钟后提醒
- [x] 随机鼓励语录，随机弹出
- [x] 右键菜单配置：显示今日提醒、添加自定义提醒、切换焦点模式、切换显示

### 备份恢复
- [x] 自动备份每次数据保存
- [x] 列出所有可用备份
- [x] 创建手动备份
- [x] 恢复选中备份
- [x] 删除备份
- [x] **导出全部数据到 Excel**，方便分析

### 友情链接
- [x] 添加/编辑/删除好友链接
- [x] 一键在浏览器打开
