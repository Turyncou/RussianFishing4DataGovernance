# RF4 数据统计与活动推荐 - PySide6 版本

这是 Russian Fishing 4 数据管理工具的 **PySide6 原生版本**，原生性能更好，界面更流畅，推荐使用。

## 📋 功能特性

### 📊 活动统计
- 多角色支持，每个角色独立统计搬砖/蹲星活动
- 自动计算今日收益、总收益、当前效率
- 支持设置目标银币/目标时长，进度条可视化展示
- 每日活动记录表格展示，可添加/删除记录
- 集成智能推荐算法，根据剩余目标计算最优每日安排
- 全局配置：每日总时长、最大并发数、切换时间开销

### 📅 每日计划
- 为每个角色的搬砖/蹲星分别设置每日目标时长
- 勾选启用/禁用特定计划
- 实时展示已完成时长和目标进度条
- 活动推荐计算**优先满足每日计划**，再分配剩余时间给剩余总目标
- 自动计算完成百分比，一目了然

### 📈 数据分析
- **每日收益趋势图** - 当日收益 + 累计收益两条曲线带动画加载
- **角色收益对比柱状图** - 各角色累计收益对比，柱状从0增长动画
- **活动时长分布图** - 饼图展示搬砖/蹲星占比，带动画展开
- **角色占比饼图** - 展示各角色总活动时长占比，超过8个自动合并"其他"
- 全部图表适配深色主题，重新加载数据/重新播放动画按钮

### 📦 存储计时
- 多个窝子/钓点累计时长追踪
- 随时增加/减少时长
- 添加/删除存储角色
- 表格展示，清晰看到剩余时长

### 🎣 饵料库存
- 记录每个饵料/拟饵/钓具的购买总量和使用总量
- 实时计算剩余数量
- 支持"增加库存"和"使用库存"操作
- 库存不足时提示确认，防止误操作
- 汇总显示总品类数和总剩余数量

### 🎡 转盘抽奖
- 自定义奖项，每个奖项可配置名称、概率、颜色
- 转盘旋转动画，物理减速模拟真实转盘
- 旋转停止自动计算中奖奖项
- 表格编辑，方便增删奖项
- 自动概率归一化，无需严格保证总概率100%

### 🔐 账号密码管理
- 多账号加密存储在本地
- 使用 **Fernet 对称加密** 算法，安全可靠
- 界面始终显示 `******`，不暴露明文和密码长度
- 一键复制密码到剪贴板，使用方便
- 加密密钥单独存储，即使数据文件泄露也无法解密

### 🔗 友情链接
- 添加/编辑/删除好友直播或网站链接
- 双击或点击按钮一键在浏览器打开
- 方便快速访问常用网站

### 🖼️ 背景设置
- 支持选择本地图片作为主窗口背景
- 可调节背景透明度（0-100%）
- 个性化你的工作台，随时更换

### 💬 桌面悬浮提醒
- 独立悬浮圆形窗口，可拖动位置
- **不显示在任务栏**，不占用任务栏位置
- **点击不抢游戏焦点** - 玩游戏全屏时点击不会切出游戏
- 气泡提示框，尖端始终紧贴圆形窗口
- 气泡浮动动画，柔和上下浮动
- **每日定时提醒**：
  - 00:00 提醒打开加班机（5分钟后二次提醒）
  - 18:00 提醒开播（5分钟后二次提醒）
  - 早/中/晚饭点提醒吃饭喝水
  - 每两小时提醒休息活动
- **自定义提醒**：支持设置 N 分钟后提醒
- **随机鼓励语录**：随机弹出鼓励话语，保持好心情

### 💾 备份恢复
- 每次数据修改自动备份
- 保留最近 10 个备份，自动轮换
- 支持手动创建备份
- 从选中备份恢复
- 删除不需要的备份
- **导出全部数据到 Excel**，方便离线分析

## 📁 数据存储位置

所有数据默认保存在用户目录下的 `.rf4_data_process/` 文件夹：

```
~/.rf4_data_process/
├── activity.json           # 活动统计数据
├── daily_tasks.json        # 每日计划配置
├── storage.json            # 存储时长数据
├── bait.json               # 饵料库存数据
├── lottery.json            # 转盘奖项配置
├── credentials.json        # 加密账号密码
├── credentials.json.key    # 加密密钥
├── friend_links.json       # 友情链接
├── background.json         # 背景设置
└── backups/                # 自动备份目录
```

首次运行自动创建目录和空数据文件。

## 🚀 运行方式

### 开发运行

```bash
# 进入qt目录
cd qt

# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

### 打包为单文件 exe

项目根目录已提供打包配置文件 `qt/rf4_data_process.spec`，直接打包：

```bash
cd qt
pyinstaller rf4_data_process.spec
```

打包完成后，exe 文件输出在 `qt/dist/` 目录，可复制到没有 Python 环境的 Windows 机器直接运行。

## ✨ 桌面提醒特性说明

### 焦点模式
- **默认：不抢焦点** - 玩游戏时点击不会夺走游戏焦点，推荐保持开启
- **抢焦点模式** - 需要弹出输入框（如添加自定义提醒）时切换到此模式，否则弹窗无法输入

### 气泡提醒特性
- 气泡尖端始终紧贴圆形窗口，无论内容多少都保持正确位置
- 气泡跟随圆形窗口一起拖动
- 文字自动居中对齐
- 所有尺寸参数都提取为类属性，方便修改适配不同大小模型
