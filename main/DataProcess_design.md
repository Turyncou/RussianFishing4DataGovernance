# DataProcess Design

## 目的

+ 使用尽可能现代化的库实现一个兼顾转盘抽奖\活动数据统计\存储时长统计\友情链接\更换图片背景等功能的程序,需要打包成.exe文件,以便于在没有python环境的电脑上运行.

## 设计

### 转盘抽奖

+ 转盘抽奖页面需要包含转盘,指针,抽奖按钮,奖项设置按钮

*** 转盘 ***: 内部根据不同奖项的概率划分不同大小的饼状区域,指针位于转盘上方固定,在点击开始抽奖按钮之后转盘开始旋转,旋转过程中指针保持不动,当转盘停止时,指针指向的区域即为抽中的奖项.同时提供奖项设置按钮,在点击后弹出新的窗口,用户可以在其中添加、删除、修改奖项以及设置每个奖项的概率.

### 活动数据统计

+ 活动数据统计包含多个角色的数据,角色支持增加和删除,每个角色包含一个数据表格,每个角色可能有两种活动，一种为搬砖，另一种为蹲星
  + 搬砖活动：表格根据活动类型包含日期、搬砖银币数量、搬砖时长等字段.用户可以在表格中直接编辑数据,也可以通过添加按钮添加新的数据行.在点击保存数据的时候会自动结算搬砖数量和搬砖时长的总和,并将数据保存到本地文件中,在结算的时候如果此时已经是第二天了,则会将前一天的数据进行归档保存,并清空当前表格中的数据,每次打开界面的时候都要判定是否已经是第二天,如果是第二天并且数据已经保存了则清空表格中的数据.同时活动数据统计界面中还有几个指标需要展示,包括今日搬砖银币数量,今日搬砖时长,总搬砖银币数量,总搬砖时长,剩余搬砖银币数量,剩余搬砖时长等,并且提供一个按钮用于指定搬砖目标,点击后弹出新窗口进行设置,只能设置计划搬砖银币数量和计划搬砖时长,在设置之后直接写回本地,在活动数据统计界面中需要根据搬砖目标和当前搬砖数据计算出完成度,并且在界面中以进度条的形式展示出来.
  + 蹲星活动：表格根据活动类型包含日期、已蹲星时长、成功蹲星数量等字段.用户可以在表格中直接编辑数据,也可以通过添加按钮添加新的数据行.在点击保存数据的时候会自动结算已蹲星时长和已蹲星数量的总和,并将数据保存到本地文件中,在结算的时候如果此时已经是第二天了,则会将前一天的数据进行归档保存,并清空当前表格中的数据,每次打开界面的时候都要判定是否已经是第二天,如果是第二天并且数据已经保存了则清空表格中的数据.同时活动数据统计界面中还有几个指标需要展示,包括今日蹲星成功数量,今日蹲星时长,总蹲星成功数量,总已蹲星时长,剩余蹲星时长等,并且提供一个按钮用于指定蹲星目标,点击后弹出新窗口进行设置,只能设置计划蹲星成功数量和计划蹲星时长,在设置之后直接写回本地,在活动数据统计界面中需要根据蹲星目标和当前蹲星数据计算出完成度,并且在界面中以进度条的形式展示出来.
  + “建议”界面：修改的本地数据使用现有的“建议”功能获取建议，并将设置建议的用户数据的按钮也添加到界面上，点击按钮弹出一个界面用于设置“建议”功能的用户数据，包括同时可以完成的A活动（搬砖）的数量和B活动（蹲星）的数量、用户切换两个活动所需时间以及用户当天想要进行活动的总时长，在活动数据统计界面的中下部分显示当前的建议，建议功能中的活动可以通过搬砖活动（A活动）和蹲星活动（B活动）的本地数据来获取，可以在这两个活动设置计划的界面中增加一个对于“收入”的设置，表明该活动整个完成时确定可以获得的收入，活动设定的目标可以切分开几次完成，不需要一次性完成整个活动，完成了多少就能获得对应比例的收入，需要的话可以微调一下“建议”功能

### 存储时长统计

+ 本功能用于存储角色存储的时长,角色支持新增和删除,并且和活动数据统计中的角色不同,整个页面包含一个数据表格,每一行代表一个角色的数据,包含剩余时长以及后置的输入框用于增加减少时长,并附带一个增加按钮一个减少按钮点击按钮后会将输入框中的数值增加或者减少到剩余时长中,并且在点击保存数据的时候会将数据保存到本地文件中,在每次打开界面的时候都要加载本地文件中的数据到表格中.

### 友情链接

+ 这个功能简单实现就行,在主界面右下角增加一个友情链接的按钮,点击后弹出一个新的窗口,窗口中展示一些友情链接,每个友情链接包含一行文字说明和一个链接地址,用户可以点击文字说明或者链接地址打开对应的链接.支持增加和删除友情链接,在增加友情链接的时候需要输入文字说明和链接地址,在删除友情链接的时候需要选择要删除的友情链接.

### 更换图片背景

+ 在主界面增加一个更换图片背景的按钮,点击后弹出一个新的窗口,用户可以选择一张本地图片,默认为预设的背景图(同一文件夹下的微信图片_20250817002102.jpg),用户可以从本地选择一张图片来更换主界面的背景图片,在选择之后会保存到本地,并且在每次打开界面的时候都要加载本地保存的背景图片,如果没有保存的背景图片则使用默认的预设背景图.同时支持透明度调整
