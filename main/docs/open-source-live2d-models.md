# 📦 开源可商用 Live2D 模型清单

这里整理了可免费使用、开源、可商用的 Live2D 模型。

---

## 🌟 官方免费示例模型（最稳定，推荐）

### 1. **Haru (小春)**
- **类型**: 短发少女，官方免费示例
- **特点**: 动作流畅，支持跟随鼠标眨眼转头，体型适中
- **格式**: Live2D v3 (`*.model3.json`)
- **下载**: [Live2D 官方示例](https://www.live2d.com/downloads/sample/samplemodel/)
- **直接下载**: https://cdn.live2d.com/editor/sample/model/Haru.zip
- **大小**: ~2.5MB

### 2. **Rice (米酱)**
- **类型**: 双马尾萝莉，官方免费示例
- **特点**: 可爱风格，多种表情动作
- **格式**: Live2D v3
- **下载**: https://cdn.live2d.com/editor/sample/model/Rice.zip
- **大小**: ~4.8MB

### 3. **Epsilon**
- **类型**: 酷酷的短发女生
- **特点**: 简洁风格
- **格式**: Live2D v3
- **下载**: https://cdn.live2d.com/editor/sample/model/Epsilon.zip
- **大小**: ~2.8MB

### 4. **Hiyori**
- **类型**: 经典看板娘，最常用的免费模型
- **特点**: 很多网站都在用，动作丰富
- **格式**: Live2D v2 (`*.model.json`)
- **下载**: https://github.com/summertriangle-dev/pylive2d/raw/master/examples/models/hiyori.zip

## 🐱 动物/宠物类模型

### 5. **Koharu 小猫**
- **类型**: 拟人化小猫娘
- **特点**: 猫耳+尾巴，非常可爱，适合做桌面宠物
- **格式**: Live2D v2
- **来源**: [PetGirl-Live2D](https://github.com/fangjyuu/PetGirl-Live2D)
- **许可**: 非商用免费

### 6. **Neco 狸猫**
- **类型**: 狸猫兽人
- **特点**: 小巧可爱，Q版风格
- **格式**: Live2D v2/v3
- **来源**: 开源社区分享

## 🔍 使用说明

### Python 集成需要注意：

| Python 库 | 支持格式 | 地址 |
|-----------|---------|------|
| **pylive2d** | Live2D v2 | https://github.com/summertriangle-dev/pylive2d |
| **live2d-py** | Live2D v3/v4 | https://github.com/chaojimaster/live2d-py |

### 推荐搭配：
- 如果你用 **pylive2d** → 选 **Hiyori** (v2)
- 如果你用 **live2d-py** → 选 **Haru/Rice/Epsilon** (v3)
