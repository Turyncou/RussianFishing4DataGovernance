## 📦 打包说明

使用 PyInstaller 打包为 Windows exe：

### 基础打包（使用系统字体）
```bash
pip install pyinstaller
cd main
pyinstaller build.spec
```

打包完成后 exe 输出在 `dist/` 目录。

### 包含自定义艺术字体打包
1. 将下载的 `.ttf` 字体文件放入 `main/fonts/` 目录
2. 修改 `desktop_reminder.py` 中字体路径为 `fonts/xxx.ttf`
3. 打包命令：`pyinstaller build.spec`（已配置自动包含 fonts 目录）

### 推荐开源可爱字体
- [站酷快乐体](https://www.zcool.com.cn/font/ZCOOLQingKeYouHappy-20190428) - 圆润活泼，推荐
- [站酷萌宠体](https://www.zcool.com.cn/font/zcoolmengchongti) - 胖胖圆圆
- [猫啃丸纸体](https://www.maoken.com/freefonts/1252.html) - 软萌手写风
