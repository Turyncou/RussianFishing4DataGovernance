# RF4 Data Process

俄罗斯钓鱼4 数据处理与桌面提醒工具，为 RF4 玩家提供数据统计和定时提醒功能。

## 功能列表

✅ **转盘抽奖** - 自定义奖项转盘抽奖
✅ **活动统计** - 多角色搬砖/蹲星数据统计，支持目标设置和进度显示
✅ **存储时长统计** - 多角色存储点卡时长剩余统计
✅ **饵料/钓具消耗统计** - 动态记录购买/使用，显示剩余库存
✅ **账号密码管理** - 本地 Fernet 加密存储，点击一键复制密码
✅ **桌面悬浮提醒** - 圆形占位（可替换为 Live2D 看板娘），定时提醒
  - 零点提醒开加班机，18:00提醒开播
  - 饭点提醒吃饭，每小时提醒休息
  - 随机鼓励话语，可自定义
  - 右键弹出可扩展菜单
  - 气泡跟随拖动，不抢游戏焦点
✅ **自动备份 & 数据导出** - 启动自动备份，支持导出 Excel
✅ **友情链接** - 自定义友情链接快速访问
✅ **更换背景** - 支持自定义背景图片和透明度
✅ **猫咪跟随** - 主界面三只猫咪会跟随鼠标移动，界面更加生动

## 如何替换为 Live2D 模型

> 当前是圆形占位符，预留了 Live2D 位置，你可以自行替换。

### 1. 准备文件

你需要准备好：
- Live2D 模型文件（通常是 `.model.json` 加上贴图 `.png`）
- 使用 Python Live2D 框架（例如 `pyLive2d` 或 `pylive2d-vub`）

推荐存放位置：
```
main/
└── live2d/
    ├── your_model/
    │   ├── model.json
    │   ├── texture_00.png
    │   └── ...
    └── （放你的模型文件夹在这里）
```

### 2. 修改代码替换

打开 `src/gui/desktop_reminder.py`，修改初始化部分：

```python
# 找到这两行（大约 35-40 行），改成你的模型大小
model_width = 250   # 根据你的模型大小调整
model_height = 400
x = screen_width - model_width - 10
y = screen_height - model_height - 10
self.window.geometry(f"{model_width}x{model_height}+{x}+{y}")
```

然后替换掉圆形绘制部分（大约 79-96 行的 `draw_circle()` 方法），改成加载 Live2D 模型即可。

### 已有的特性会自动继承：

| 特性 | 是否自动生效 |
|------|-------------|
| 始终置顶显示 | ✅ 已配置 |
| 点击不抢游戏焦点 | ✅ 默认开启，可切换 |
| 拖动限制在屏幕内 | ✅ 自动生效 |
| 气泡跟随拖动 | ✅ 自动生效 |
| 右键弹出菜单 | ✅ 已实现 |
| 动态大小气泡 | ✅ 已实现 |
| 气泡文本居中 | ✅ 已实现 |

也就是说，你只需要替换渲染部分为 Live2D，所有交互和提醒功能都不用改！

## 运行方式

### 开发运行

```bash
# 安装依赖
pip install -r ../requirements.txt

# 运行程序
cd main
python main.py
```


打包后的exe文件会出现在 `dist/` 目录中，可以直接在没有Python环境的电脑上运行。

## 详细功能说明

### 🎡 转盘抽奖
- 点击"开始抽奖"按钮转盘开始旋转，停止后指针指向即为中奖
- 点击"奖项设置"可以添加、删除、修改奖项，设置每个奖项的概率和颜色
- 总概率不需要严格等于100%，程序会按比例计算

### 📊 活动统计（搬砖/蹲星）
- 支持多个角色，每个角色独立统计
- 自动统计今日和总计银币、时长
- 支持设置目标银币和目标时长，显示进度条
- 自动计算当前效率，并给出完成天数建议
- 每天打开程序时，只会保留当日数据，历史数据自动归档

### 📦 存储时长统计
- 支持多个存储角色，显示剩余时长
- 点击选择角色后，输入分钟数，可以增加或减少时长
- 数据自动保存到本地文件

### 🎣 饵料/钓具消耗统计
- 动态管理饵料、拟饵、钓具库存
- 记录购买总量和使用量，自动计算剩余
- 支持随时添加、删除、调整

### 🔐 账号密码管理
- 多个游戏账号加密存储在本地
- 使用 Fernet 对称加密，密钥也存在本地
- 下拉框选择账号，密码显示 `******`，点击一键复制到剪贴板

### 💬 桌面悬浮提醒
- **占位设计**：当前是圆形占位，未来可替换为 Live2D 看板娘
- **定时提醒**：
  - 00:00 提醒打开加班机（5分钟后二次提醒）
  - 18:00 提醒开播（5分钟后二次提醒）
  - 早中晚饭点提醒吃饭喝水
  - 每两小时提醒休息活动
- **随机鼓励**：15-30分钟随机弹出鼓励话语，内容可在代码开头自定义
- **交互特性**：
  - 左键拖动位置，限制在屏幕内不会拖出去
  - 双击隐藏/显示
  - 右键弹出可扩展菜单
  - 气泡跟随拖动
  - **默认不抢焦点**：点击操作不会夺走游戏焦点，游戏全屏不受影响
  - 可切换点击模式和气泡置顶选项

### 🔗 友情链接
- 点击右下角"友情链接"打开
- 可以添加、删除链接
- 双击链接或选中后点"打开链接"会在浏览器中打开对应网址

### 🖼️ 更换背景
- 点击左下角"更换背景"打开
- 选择本地图片作为背景，可以调整透明度
- 默认背景是 `微信图片_20250817002102.jpg`

### 💾 自动备份
- 每次启动程序自动备份所有数据文件
- 保留最近 10 个备份，自动轮换
- 支持手动从备份恢复，支持导出全部数据为 Excel

## 数据存储

所有数据保存在用户目录下的 `.rf4_data_process/` 文件夹中：
- `lottery.json` - 转盘奖项设置
- `activity.json` - 活动统计数据
- `storage.json` - 存储时长数据
- `bait.json` - 饵料消耗数据
- `credentials.json` - 加密的账号密码
- `credentials.json.key` - 加密密钥
- `friend_links.json` - 友情链接
- `background.json` - 背景设置
- `backups/` - 自动备份目录

## 桌面提醒特性说明

### 焦点模式（默认不抢焦点）
- **✅ 不抢焦点模式（推荐玩游戏时使用）**：点击拖动模型、打开菜单都不会夺走游戏窗口焦点，游戏全屏完全不受干扰
- **❌ 抢焦点模式**：需要打开自定义提醒对话框时切换到此模式，否则弹窗可能无法输入

### 气泡置顶
- **✅ 始终置顶**：气泡永远显示在最前面
- **❌ 不强制置顶**：气泡可能被其他窗口覆盖

### 拖动边界限制
- 模型窗口始终限制在屏幕范围内，不会拖出去找不到
