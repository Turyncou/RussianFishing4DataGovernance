# RF4 Data Process

俄罗斯钓鱼4 数据处理工具，包含以下功能：

- **转盘抽奖** - 自定义奖项转盘抽奖
- **搬砖数据统计** - 多角色搬砖数据统计，支持目标设置和进度显示
- **存储时长统计** - 多角色存储点卡时长剩余统计
- **友情链接** - 自定义友情链接快速访问
- **更换背景** - 支持自定义背景图片和透明度
- **猫咪跟随** - 主界面三只猫咪会跟随鼠标移动，界面更加生动

## 运行方式

### 开发运行

```bash
# 安装依赖
pip install -r ../requirements.txt

# 运行程序
cd main
python main.py
```

### 打包为exe

```bash
pyinstaller build.spec
```

打包后的exe文件会出现在 `dist/` 目录中，可以直接在没有Python环境的电脑上运行。

## 功能说明

### 转盘抽奖
- 点击"开始抽奖"按钮转盘开始旋转，停止后指针指向即为中奖
- 点击"奖项设置"可以添加、删除、修改奖项，设置每个奖项的概率和颜色
- 总概率不需要严格等于100%，程序会按比例计算

### 搬砖数据统计
- 支持多个角色，每个角色独立统计
- 自动统计今日和总计银币、时长
- 支持设置目标银币和目标时长，显示进度条
- 自动计算当前效率，并给出完成天数建议
- 每天打开程序时，只会保留当日数据，历史数据自动归档

### 存储时长统计
- 支持多个存储角色，显示剩余时长
- 点击选择角色后，输入分钟数，可以增加或减少时长
- 数据自动保存到本地文件

### 友情链接
- 点击右下角"友情链接"打开
- 可以添加、删除链接
- 双击链接或选中后点"打开链接"会在浏览器中打开对应网址

### 更换背景
- 点击左下角"更换背景"打开
- 选择本地图片作为背景，可以调整透明度
- 默认背景是 `微信图片_20250817002102.jpg`

## 数据存储

所有数据保存在用户目录下的 `.rf4_data_process/` 文件夹中：
- `lottery.json` - 转盘奖项设置
- `grinding.json` - 搬砖数据
- `storage.json` - 存储时长数据
- `friend_links.json` - 友情链接
- `background.json` - 背景设置

## 技术栈

- Python
- CustomTkinter - 现代化GUI界面
- Pillow - 图片处理
- PyInstaller - 打包为exe
